``quantlib``
============

The ``quantlib`` package is the heart of QuantLab.
In this package you can find:
* the implementations of multiple quantization algorithms (``algorithms`` sub-package);
* the implementations of graph analysis, editing, and morphing functionalities (``graphs`` sub-package).
